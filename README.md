# Wyvern Hyperspectral Project
-------------------------------------------

## What is it?
This is a repo for the simulated Dragonette image product pipeline. It takes in
1m ground sample distance (GSD) aerial imagery with 426 bands and converts it to
5m GSD imagery with 18 bands. This is to simulate the capabilities of the
Dragonette spacecraft.

## Main Features
- Spatial resampling
- Spectral resampling
- STAC Json metadata output
- Geotiff image output
- Preview and thumbnail image output
